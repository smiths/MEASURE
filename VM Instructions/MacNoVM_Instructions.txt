Mac Users

1. Install Office 365

2. Login to Vena using the Google Chrome browser (other browsers may not provide all of the necessary functionality.)

3. Checkout the rubric, which will save it locally.

4. Insert/Add-Ins/Store, search for and install the Vena add-in

5. Edit the local version of the rubric.

6. Save the rubric using the Vena Add-in
